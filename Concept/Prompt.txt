Build a full-stack production-ready web application named "Nyxecure".

## 🛡️ PROJECT OVERVIEW

Nyxecure is a cybersecurity platform that combines:

The application must feel like a modern SaaS cybersecurity dashboard, not a student project.

## 🌐 PHISHING URL DETECTOR

Frontend:

* Input field for URL
* Button: "Analyze URL"

Backend logic:

* Check HTTPS presence
* Detect suspicious keywords:
  ["login", "verify", "bank", "secure"]
* Count dots and URL length
* Optional: Google Safe Browsing API integration

Output:

* Status:

  * Safe (green)
  * Suspicious (yellow)
  * Dangerous (red)
* Risk score (0–100)
* Explanation list
